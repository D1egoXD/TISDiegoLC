Clientes

Campos (tabla Clientes)

  • ID Cliente
  • Nombre
  • Correo
  • Teléfono
  • Dirección
  • Sexo
  • Fecha de nacimiento
  • Edad

Roles y Permisos

Supervisor: registrar, consultar y actualizar la información de los clientes para mantener control organizado y confiable.

Empleado (Ingeniero Civil / Supervisor): consultar datos generales y actualizar datos de contacto únicamente.

Objetivo

Mantener un registro organizado de los clientes para un mejor seguimiento y comunicación. Garantizar que la información se mantenga actualizada y accesible.

Trabajadores

Campos (tabla Trabajadores)

  • ID Trabajador
  • Nombre
  • Correo
  • Teléfono
  • Dirección
  • Sexo
  • Fecha de nacimiento
  • Puesto

Roles y Permisos

Supervisor: administrar datos (registro, consulta, actualización).

Empleado (Ingeniero Civil): consultar información; actualizar datos de contacto (teléfono y dirección).

Objetivo

Asegurar administración clara del personal y mantener contactos actualizados para coordinación en proyectos.

Planos

Campos (tabla Planos)

  • ID Plano
  • Código
  • Calle
  • Número
  • Proyecto asociado
  • Encargado

Roles y Permisos

Arquitecto: registrar, consultar, actualizar y eliminar planos.

Supervisor: registrar, consultar, actualizar y eliminar planos.

Ingeniero: consultar y actualizar planos.

Objetivo

Asegurar control integral y disponibilidad de la información técnica para evitar errores en obra.

Proveedores

Campos (tabla Proveedores)

  • ID Proveedor
  • Nombre
  • Correo
  • Teléfono
  • Dirección
  • Sexo
  • Fecha de nacimiento
  • Bloquear (flag)

Roles y Permisos

Supervisor: registrar, consultar, actualizar (correo, teléfono, dirección) y bloquear proveedores por ID.

Empleado (Ingeniero): consultar y actualizar datos básicos de proveedores.

Objetivo

Garantizar control preciso de proveedores para asegurar disponibilidad de insumos y eficiencia en compras.

Área de Finanzas

Campos (tabla Finanzas)

  • ID Registro
  • Responsable
  • Correo
  • Teléfono
  • Presupuesto asignado
  • Gastos
  • Facturas

Roles y Permisos

Gerente: revisar presupuestos, modificar gastos y eliminar registros obsoletos.

Empleado (Contador): registrar facturas y actualizar gastos.

Objetivo

Mantener control detallado de ingresos y egresos para garantizar estabilidad financiera.

Área Legal

Campos (tabla Legal)

  • ID Documento
  • Tipo de contrato
  • Cliente/Proveedor asociado
  • Fecha inicio
  • Fecha vencimiento
  • Estado

Roles y Permisos

Gerente: registrar contratos, actualizar fechas y eliminar contratos.

Empleado (Abogado): consultar y actualizar estado de contratos.

Objetivo

Tener contratos y permisos actualizados para cumplir con la ley.

Permisos y Licencias

Campos

  • ID Licencia
  • Tipo
  • Fecha emisión
  • Fecha vencimiento
  • Proyecto asociado

Roles y Permisos

Gerente: registrar, actualizar y eliminar licencias.

Empleado (Abogado): consultar y actualizar datos de licencias.

Objetivo

Garantizar que todos los proyectos cuenten con permisos vigentes.

Seguros

Campos

  • ID Seguro
  • Tipo (obra, personal, maquinaria)
  • Fecha inicio
  • Fecha vencimiento
  • Estado

Roles y Permisos

Gerente: registrar, actualizar y eliminar seguros.

Empleado (Gerente de seguros): consultar seguros y estados.

Objetivo

Proteger a la empresa, personal y maquinaria de riesgos.

Almacén (Materiales y Maquinaria)

Campos

  • ID Producto/Maquinaria
  • Nombre
  • Tipo (material / maquinaria)
  • Cantidad
  • Estado
  • Ubicación
  • Proveedor

Roles y Permisos

Gerente: registrar, actualizar cantidades/estado y eliminar registros.

Empleado (Supervisor / Ingeniero Civil): consultar y actualizar cantidades.

Objetivo

Controlar el inventario para evitar pérdidas y asegurar disponibilidad.

Transporte

Campos

  • ID Vehículo
  • Tipo
  • Capacidad
  • Estado
  • Responsable
  • Ubicación

Roles y Permisos

Gerente de transporte: registrar vehículos, actualizar estado y eliminar datos.

Objetivo

Administrar el uso de vehículos y maquinaria de transporte.

Ingeniero en Seguridad e Higiene

Campos

  • ID Ingeniero
  • Nombre
  • Proyecto
  • Reportes de seguridad

Roles y Permisos

Gerente: asignar proyectos y eliminar registros.

Empleado (Ingeniero de seguridad): registrar y actualizar reportes.

Objetivo

Controlar inspecciones de seguridad en la obra.

Ingenieros Especialistas (Eléctrico, Mecánico, Hidráulico)

Campos

  • ID Ingeniero
  • Nombre
  • Especialidad
  • Proyecto
  • Contacto

Roles y Permisos

Supervisor: asignar proyectos y eliminar registros.

Empleado (Ingeniero especialista): actualizar contacto y avances del proyecto.

Objetivo

Tener organizados a los ingenieros por especialidad para cada obra.

Diseñador de Interiores / Paisajista

Campos

  • ID Diseñador
  • Nombre
  • Proyecto
  • Diseño asociado

Roles y Permisos

Diseñador: asignar proyectos y eliminar registros.

Empleado (Diseñador): registrar y actualizar diseños.

Objetivo

Mantener diseños estéticos y paisajísticos organizados por proyecto.
